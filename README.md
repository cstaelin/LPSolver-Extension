**LPSolver 4.1.1**

LPSolver uses the lpsolve55 linear programming library to solve linear programs.  It can handle normal, integer and mixed integer problems of any dimension.  The extension is described in the PDF file, *LPSolver Extension-v4.1.pdf*, distributed with the extension.  Instructions for installing the extension on on Windows, the Macintosh and Linux are contained in the file *INSTALLATION.md*.  Licensing information can be found in the file *LICENSE.md*.  The required dynamic libraries for each platform are distributed with the extension, as are the Java source files.

NOTE THAT THIS VERSION WORKS ONLY ON 64-BIT OPERATING SYSTEMMS. Future versions will include 32-bit systems and the Macintosh x86 platform.

Questions about this version of the LPSolver extension can be directed to Charles Staelin at the [github issue tracker](https://github.com/cstaelin/LPSolver-Extension/issues) for this extension.
